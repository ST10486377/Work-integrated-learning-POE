The Empowering the Nation project is designed to promote skills development and accessible education through both a website and a mobile application.
These platforms provide information about accredited training programs that prepare individuals for employment and entrepreneurship.
The system allows users to explore available courses, calculate total fees, and contact the organization for enrollment.

⸻

1. Website Functionality

The web platform provides an easy-to-navigate interface with the following main sections:
	•	Home Page: Introduces the organization and provides quick access to all sections.
	•	Courses Page: Lists available 6-week and 6-month accredited training programs (e.g., Cooking, Baking, Sewing, Life Skills).
	•	Fee Calculator: Lets users select one or more courses and view the total estimated cost, including VAT.
	•	Contact Page: Provides details such as phone, email, and campus locations for inquiries or registration.

The site’s design emphasizes simplicity, accessibility, and responsiveness, ensuring it works smoothly across desktops, tablets, and mobile browsers.

⸻

2. Mobile App Functionality

The mobile app offers the same core features as the website, optimized for smartphones and smaller screens.
Key features include:
	•	A simplified interface for easy navigation using buttons and icons.
	•	Course browsing by category or duration.
	•	An integrated fee calculator for mobile use.
	•	A quick-access contact section for calls or emails.

The mobile version ensures accessibility for users with limited digital experience and low data connectivity.

⸻

3. How to Use
	1.	Open the website in your preferred browser or install the mobile app.
	2.	From the home screen, choose between 6-week or 6-month courses.
	3.	Use the Fee Calculator to estimate your total cost.
	4.	Visit the Contact Page to reach a nearby campus or submit a question.
	5.	Enroll directly or request additional information.

⸻

4. Technical Details
	•	Languages Used: HTML, CSS, and basic responsive design principles.
	•	Design Tools: Canva and Figma (used for logo and wireframes).
	•	Testing: The interface was tested for compatibility on Android devices and common web browsers.
	•	Design Theme: Light color palette (blue and orange) representing trust, optimism, and empowerment.

6. Summary

The Empowering the Nation web and mobile applications aim to make education more accessible to South Africans through digital technology.
By integrating usability, affordability, and functionality, the project empowers users to take control of their learning and future opportunities.
